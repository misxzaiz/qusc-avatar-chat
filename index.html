📖 使用说明

🔑 DeepSeek API 配置

1

获取 API 密钥

访问 DeepSeek 官网 注册账号

2

创建 API Key

登录后前往 API 管理页面，创建新的 API 密钥

3

配置密钥

点击右上角 ⚙️ 设置按钮，在 "DeepSeek API Key" 中输入您的密钥

4

开始聊天

保存设置后即可开始与 AI 对话！

💡 温馨提示

  • 请妥善保管您的 API 密钥，不要泄露给他人
  • 首次使用建议先测试简单对话确认配置正确
  • 如遇到网络问题，请检查网络连接和防火墙设置

💬 智能对话

  • 实时对话：支持流式对话，实时显示 AI 回复
  • 上下文理解：AI 会记住对话历史，提供连贯回复
  • 多轮对话：支持长时间深度交流

🎤 语音功能

  • 语音输入：点击 🎤 按钮进行语音识别输入
  • 语音输出：AI 回复可转换为语音播放
  • 语音设置：可调节语速、音调、音量等参数
  • 语音打断：录音时自动停止当前播放

😊 悬浮表情助手

  • 启动助手：点击 😊 按钮启动悬浮表情窗口
  • 三种模式：
    • 🎭 卡通表情：可爱的字符表情，支持多种情绪
    • 🖼️ 自定义图片：上传您喜欢的图片作为头像
    • 🎬 自定义视频：上传动态视频作为头像
  • 智能反应：根据对话状态自动变换表情
  • 尺寸调节：点击 📏 调整窗口大小
  • 窗口操作：支持拖拽移动、最小化、关闭等操作

👤 角色管理

  • 角色切换：点击 👤 管理和切换 AI 角色
  • 预设角色：提供多种预设角色（助手、老师、程序员等）
  • 自定义角色：创建专属的 AI 角色
  • 快速生成：通过关键词快速生成角色

🎨 个性化设置

  • 聊天背景：支持纯色、图片、视频背景
  • 透明度调节：调整消息框和背景透明度
  • 历史记录：点击 📚 查看聊天历史
  • 移动端优化：完美支持手机和平板设备
